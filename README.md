# moni

Helps humans keep track of their spendings.

## install|run
1. clone the repo
2. make a virtualenv to run the webapp
```bash
virtualenv -p python3 moni-env
```
3. install the dependencies
```bash
pip install -r requirements
```
4. init the database
```bash
flask init-db
```
5. run the app
```bash
flask run
```
